Instantly generate random numbers with our customizable Number Generator. Set specific ranges, allow or prevent duplicates, and sort your results with ease.

[https://www.calculator.io/number-generator/](https://www.calculator.io/number-generator/)

== Usage ==

Add the Number Generator shortcode to your page, post or sidebar:

`[calcio_number_generator]`

Add the following code to your template where you would like the Number Generator to appear:

`<?php display_calcio_number_generator(); ?>`

== Installation ==

1. Upload the Number Generator /calcio_number_generator/ folder to the /wp-content/plugins/ directory.
2. Activate the Number Generator plugin through the "Plugins" menu in WordPress.
